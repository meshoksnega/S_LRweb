How to Change Faces

1

Select a Face

Select a face in the Face collection or directly in the viewport

2

Draw a Face

Draw a face (resolution 18×18 px). Save faces in a dedicated folder named Face0, Face1, Face2, etc.

3

Switch Faces

Open the Shader Editor, import multiple faces with Shift/Ctrl, use Offset slider to switch faces
