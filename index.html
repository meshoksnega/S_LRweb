How to Change Faces

01

Select a Face

Select a face in the Face collection or directly in the viewport to begin the customization process.

02

Draw a Face

Create your custom face with the required resolution of 18x18 pixels. Save your faces in a dedicated folder, naming them Face0, Face1, Face2, etc.

03

Import & Switch

Open the Shader Editor. Click the folder icon and import your face textures using Shift or Ctrl for multiple selections. Use the Offset slider to switch between faces.
